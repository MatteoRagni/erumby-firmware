** Class for the SERVO (Servo motor controller)**

The class implements the software representation of the Servo motor Controller of the car. It takes a PWM as input and sets a voltage on the servo input in order to get movement. The SERVO class also implements some security features such raising an alarm when the input is out the boundaries.

The user should be aware that for the current hardware the full right control is the minimum value in terms of PWM while the full left control is the maximum value in terms of PWM. The code handles the opposite situation at compile time

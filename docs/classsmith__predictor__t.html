Detailed Description

template<timing_t MILLIS, timing_t DELAY>
class smith_predictor_t< MILLIS, DELAY >

Smith predictor.

An implementation of a smith predictor for the following linear plant:

\begin{align} \dot{x}(t) & = - a x(t) + a \mathrm{sat}(u(t - d)) \\ y(t) & = \phi(x(t)) \end{align}

This class is taylored made for our ESC system. The integration of this system is performed with a Backward Euler \( s = \frac{z-1}{t_s z} \) and the recursion follows:

\begin{align} q_{k} & = \mathrm{sat}_{[0,1]}(t_{k}) \\ x_{k} & = a_{sp} x_{k-1} + b_{sp} q_{k} \\ y_{k} & = \phi(x_{k}) = \frac{\sqrt{c_1^2 + 4 c_2 x_{k}} - c_1}{2 c_2} \end{align}

where:

\begin{align} a_{sp} = (1 + a t_s)^{-1} b_{sp} = a_{sp} a t_s \end{align}

The actual output in the control loop, for a discretization of the delay in a cyclic array with size \( n = \mathrm{floor}(d / t_s) \), follows:

\begin{align} \omega &= y_{k - n} \\ \omega_{predict} &= y{k} \end{align}

Warning
The non linearity is a virtual method. Thus it should be redefined in the controller.

Detailed Description

template<typename T, size_t N>
class cyclic_array_t< T, N >

Cyclic array implementation.

The Cyclic array implementation allows to have a queue of elements for which the user can push back elements, but without using allocation.

Template Parameters
Ttype contained in the internal array
Nsize of the internal array

inlineprivate

Returns the data index from the rolling index.

The functions evaluates the actual position in data by using a remainder operation. If on AVR (e.g. Arduino) the function returns an index even if the parameter is out of bound. For compilers capable of handling exceptions, it will raise an std::out_of_bound exception for index greter than size.
